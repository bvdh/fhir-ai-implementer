# FHIR-34332: Request to include additional search parameters in the registry

## Metadata

- **Assignee**: Unassigned
- **Created**: 11/18/21
- **Description**: The search parameter registry does not have the complete list of search parameters. Only few params in the section - Parameters for all resources and Search  result parameters are listed in the registry such as _id, _lastUpdated, _content, _profile, _query, _security, _source and _tag. Other list of parameters in the aforementioned section also need to be added to search registry. Likewise, special search parameters also should be included. These are: _query, _filter, _format and _pretty.

Although in my opinion all params should be part of the registry, however wanted to know if there is any reason on why only few added initially?
