# FHIR-48778: meta.tag is example bound to common-tags

## Metadata

- **Assignee**: Unassigned
- **Created**: 11/6/24
- **Description**: the meta.tag element is example bound to common-tags, which is defined as experimental.

Yet, this valueset does include two codes that have conformance language about their use. 

Seems this should be required binding to an active valueSet.
- **Issue Type**: Change Request
- **Key**: FHIR-48778
- **Related Artifact(s)**: BaseResource
- **Reporter**: John Moehrke
- **Resolution**: Persuasive with Modification
- **Resolution Description**: This can't be required.  It should ideally be extensible, but that would be a breaking change to a normative structure.

We will mark the value set and code system as NOT experimental.

We will change the binding strength from 'example' to 'preferred'.
- **Status**: Applied
- **Summary**: meta.tag is example bound to common-tags
- **Work Group**: FHIR Infrastructure
