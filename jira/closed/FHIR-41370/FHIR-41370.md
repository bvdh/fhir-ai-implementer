- **Description**: The FHIR specification defines the regex for date, datetime and time but not the calendar used. 

FHIRpath specifies that ISO 8601 is used. Which was my assumption as well for the FHIR specification (as FHIR uses FHIRpath). But I could not fined a reference to ISO8601 in the FHIR specification.

I think this is an oversight that should be corrected by adding a reference to ISO8601 to date, datetime and time.
