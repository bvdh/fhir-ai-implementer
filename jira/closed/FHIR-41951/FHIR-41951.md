# FHIR-41951: Add Medication and Substance to Group.member.entity

## Metadata

- **Assignee**: Unassigned
- **Created**: 8/25/23
- **Description**: It is possible to do tests on tablets or on raw materials (Observation.subject can be Medication or Substance). You can also have these as the subject of a DiagnosticReport.

However typically you would test several tablets, so, since DiagnosticReport.subject is 0..1 that means using Group. But then you cannot put the Medication or Substance as the member of a group.

Adding Medication and Substance to Group.member.entity would address this.
- **Issue Type**: Change Request
- **Key**: FHIR-41951
- **Related Artifact(s)**: Group
- **Related URL**: http://build.fhir.org/group.html
- **Reporter**: Rik Smithies
- **Resolution**: Persuasive with Modification
- **Resolution Description**: Do this.  Also:
 * add BiologicallyDerivedProduct and NutritionProduct.
 * add 'medication' and 'substance', 'biologicallyDerivedProduct' and 'nutritionProduct' to the list of codes for Group.type.
 * if necessary, update the target resources of the member search parameter to include the new resources
- **Status**: Applied
- **Summary**: Add Medication and Substance to Group.member.entity
