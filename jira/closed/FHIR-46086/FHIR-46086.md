- **Description**: FHIRPath Patch allows for a number of specified codes for the available operation types (add, insert, delete etc). It would be good if these codes were listed in a ValueSet.

https://build.fhir.org/fhirpatch.html

In that way it would be possible to make your own ValueSets containing subsets of the list of codes. This makes it easier for actors using the operation type codes to specify which codes are supported in different scenarios.
