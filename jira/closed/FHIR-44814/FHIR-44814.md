- **Description**: In the Environment Variables section the examples use the " (double quote) character to delimit a variable name that has reserved characters where fhirpath itself defines ` (backtic)

%"vs-&#91;name&#93;" // (string) full url for the provided HL7 value set with id &#91;name&#93;
%"ext-&#91;name&#93;" // (string) full url for the provided HL7 extension with id &#91;name&#93; 

should be

%`vs-&#91;name&#93;` // (string) full url for the provided HL7 value set with id &#91;name&#93;
%`ext-&#91;name&#93;` // (string) full url for the provided HL7 extension with id &#91;name&#93;

and 
Observation.component.where(code.memberOf(%`vs-observation-vitalsignresult`))
and
%`us-zip` = '&#91;0-9&#93;{5}(-&#91;0-9&#93;{4}){0,1}'
 
Also, in the factory section the example for creating a Coding shows calling the Quantity function
 
Creates a Coding: &lt;code&gt;%factory.Quantity('http://loinc.org', '1234-5, 'An example test', '1.02')&lt;/code&gt;
That should read Coding
