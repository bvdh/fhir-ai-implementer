# FHIR-44925: Add some clarifications to the trademark license information

## Metadata

- **Assignee**: Unassigned
- **Created**: 3/8/24
- **Description**: After some discussion about the use of the FHIR trademark (https://chat.fhir.org/#narrow/stream/179166-implementers/topic/Trademark.20question.20about.20text.20to.20include) it seemed to me that there were a few helpful points that could be added this documentation that would help clarify things:

1) Grahame's description of what a "trademark use" is was very helpful.  The current text mentions what to do for a "non-trademark use" without making clear what those two categories are.

2) Grahame's point that the trademark restrictions apply even to the title of a slide deck was surprising and would be useful to include

3) Lloyd's example of the title, "FHIRing on All Cylinders" as one needing permission would be good to include.

4) Some other examples could useful as well.  Here are some I think would be helpful (though I am not sure on which side of fence these land):

a) "CompanyX's FHIR Products" as a website title.

b) The hostname "companyx-fhir-products" for the same website.

c) "CompanyX's Products for FHIR" as a website title.
