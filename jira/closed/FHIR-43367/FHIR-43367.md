# FHIR-43367: Example for extensions for inter-version extensions

## Metadata

- **Assignee**: Unassigned
- **Created**: 12/12/23
- **Description**: The example of how to do inter version extensions using a ValueSet resource includes an R4 version that has elements that do not exist in the FHIR R4 ValueSet resource.

The example below was listed as a R4 ValueSet resource, but in that resource ValueSet.expansion.contains does not contain the element "property".


{
  "resourceType" : "ValueSet",
  "expansion" : {
    "contains" : [{
      "property" : [{
        "code" : "prop-code",
        "valueString" : "prop-value",
        "extension" : [{
          "url" : "http://hl7.org/fhir/5.0/StructureDefinition/extension-ValueSet.expansion.contains.property.subProperty",
          "extension" : [{
            "url" :"code",
            "valueCode" : "sub-prop-code"
          },{
            "url" : "value",
            "valueCode" : "sub-prop-value"
          }]
        }]
      }]
    }]
  }
} 

Change the example to make the "property" element an inter version extension or change to a different example.
