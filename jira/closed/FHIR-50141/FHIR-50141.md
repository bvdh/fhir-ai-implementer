# FHIR-50141: Questionnaire.item.enableWhen.answer[x] does not support URL (valueUri) type

## Metadata

- **Assignee**: Unassigned
- **Created**: 3/28/25
- **Description**: Missing Spec

Questionnaire.item.enableWhen is supposed to support all valid FHIR R4 question types, however, the Url type (valueUri) is missing from the enableWhen spec.

Questionnaire.item, QuestionnaireResponse.item, and more R4 Backbone Elements all support type 'Url' questions.  Missing support in Questionnaire.item.enableWhen breaks software implementations for conditionally rendering and disabling elements.

 

e.g. NLM Form Builder when referencing a Url item for conditional display

https://formbuilder.nlm.nih.gov/

 

Resolution Request

Please add valueUri to the FHIR R4 Questionnaire.item.enableWhen spec, as well as the @types/fhir/r4 Typescript repository (https://www.npmjs.com/package/@types/fhir)
