- **Description**: The expectations of what the Coding is for within a CodeableConcept is documented, as well as the purpose of using multiple Codings (i.e., translations, etc) but the expectation that a Coding should not contradict other Codings within the same CodeableConcept is not addressed in any way.

We need to provide some guidance on this expectation of non-contradicting Codings in the specification.
