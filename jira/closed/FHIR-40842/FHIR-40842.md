# FHIR-40842: Incorrect description of :code-text on search page

- **Description**: Per the resolution of FHIR-32673, :code-text is to "allow partial text searching on the code or value portion of the Coding or Identifier".

However, the brief description of :code-text on Search - FHIR v5.0.0 (hl7.org) says, "Tests whether the textual display value in a resource (e.g., CodeableConcept.text, Coding.display, or Reference.display) matches the supplied parameter value." - which is very nearly the same as :text, i.e., it is matching against the text portion of Coding rather than the code.

The detailed description of :code-text at Search - FHIR v5.0.0 (hl7.org) correctly says, "This modifier is used to do a 'standard' string search against code values."
