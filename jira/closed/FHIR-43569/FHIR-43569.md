- **Description**: The MedicationRequest status codes include ended and https://hl7.org/fhir/sc-valueset-medicationrequest-status.html tell that it is equivalent/mapping to ended in Canonical Status Codes for FHIR Resources

but ended status does not exist in the canonical status table https://hl7.org/fhir/valueset-resource-status.html
so what is the equivalent of ended of Medication Request status?
- **Issue Type**: Change Request
- **Key**: FHIR-43569
- **Related Artifact(s)**: MedicationRequest
- **Related URL**: https://hl7.org/fhir/codesystem-resource-status.html#resource-status-error
- **Reporter**: AhmedEl-Tabarani
- **Resolution**: Persuasive with Modification
- **Resolution Description**: Add {{ended}} as a code with the description "The process described/requested has reached a state where no further action is planned".

{{ended}} should be inserted such that it has children {{failed}}, {{complete}}, and {{abandoned}}.
