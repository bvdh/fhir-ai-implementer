- **Description**: The ImplementationGuide-example.json contains a "globals" element:

  

It causes a validation error when using FHIR validator to validate a patient resource using local package.tgz. 



java -jar validator.jar -ig us_core_package.tgz patient.json



The validator warning is:



  Warning @ Patient (line 1, col2): Global Profile reference 'http://hl7.org/fhir/us/core/StructureDefinition/patient' from IG http://hl7.org/fhir/us/daf|0 could not be resolved, so has not been checked



FHIR validator does not return the above warning when using package-id as parameter.
