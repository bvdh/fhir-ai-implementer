# FHIR-45488: The curating work group for the Data types and Metadata Types pages should be FHIR-I and not MnM

## Metadata

- **Assignee**: Unassigned
- **Created**: 5/6/24
- **Description**: The stated curating work group for these pages is currently Modeling and Methodology (MnM) which is no longer an active work group. The curating work group should be changed to FHIR Infrastructure (FHIR-I).

Relatedly, please note the following editorial inconsistency:

The title of the Metadata Types page is:

	MetaDataTypes in the section heading (alongside the section number); and
	MetaData Types in the navigation links (alongside Foundation &gt; Document Index &gt; Datatypes)



It would be good if these were changed to be the same.
