- **Description**: Each SubscriptionTopic resource defines a set of allowed filters (SubscriptionTopic.canFilterBy), which can be referred to within a Subscription resource (Subscription.filterBy).
The Subscription not just refers to the filter but is also using it to tune the notification content. I suggest to replace it with:
Each SubscriptionTopic resource defines a set of allowed filters (SubscriptionTopic.canFilterBy), which can be used within a Subscription resource (Subscription.filterBy) to restrict the notifications provided by the SubscriptionTopic.
