- **Description**: Spec on how to handle references to additional resources did not make the spec. This was discussed in the Sept 2025 WGM and was voted on. As documented in the notes that can be found at:  https://confluence.hl7.org/spaces/FHIRI/pages/358879917/FHIR+Infrastructure+Minutes+WGM+202509+-+Pittsburgh#FHIRInfrastructureMinutesWGM202509Pittsburgh-WedQ3

 

The intended change was that references to additional resources can be made using normal references if this is declared in the StructureDefinition of the additional resource.

This relates to section:

https://build.fhir.org/references.html#additional

 

Gino's has a transcript of this discussion which will be added to this ticket.
