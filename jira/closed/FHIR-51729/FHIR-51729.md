# FHIR-51729: Remove unused elements from CompartmentDefinition resource

## Metadata

- **Assignee**: Unassigned
- **Created**: 8/12/25
- **Description**: None of the defined compartments use the documentation, {{startparam }}or {{endparam }}elements.

CompartmentDefinitions are only permitted to be defined by HL7. Therefore, our defined compartments are the complete (current set) of compartmentdefinitions, and represent all usage of the resource. Since these elements are not exercised in our definitions, they are not used anywhere.

Remove the unused elements.
- **Issue Type**: Change Request
- **Key**: FHIR-51729
- **Related Artifact(s)**: CompartmentDefinition
- **Reporter**: Elliot Silver
- **Resolution**: Persuasive with Modification
- **Resolution Description**: We will remove the .documentation element.  While technically a breaking change, because *only* HL7 can define instances, and none of our instances use the element, it's not breaking in practice and therefore acceptable as a change to a normative artifact.

We will update the existing compartment definitions to make use of the start and end param elements.
- **Status**: Applied
- **Summary**: Remove unused elements from CompartmentDefinition resource
- **Work Group**: FHIR Infrastructure
