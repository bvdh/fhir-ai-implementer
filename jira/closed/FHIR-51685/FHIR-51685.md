# FHIR-51685: Need to search StructureMap by from/to SDs

## Metadata

- **Assignee**: Unassigned
- **Created**: 8/7/25
- **Description**: It should be possible to search for StructureMaps that map to a given StructureDefinition or from a given StructureDefinition, and right now there aren't standard search parameters to do this.
- **Issue Type**: Change Request
- **Key**: FHIR-51685
- **Related Artifact(s)**: StructureMap
- **Reporter**: Lloyd McKenzie
- **Resolution**: Persuasive with Modification
- **Resolution Description**: We will define search parameters for source, target, queried, and produced, with filters of structure.where(mode='one of the above').url with a search type of url.
