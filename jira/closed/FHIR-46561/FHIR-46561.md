# FHIR-46561: Rendering should display resource restrictions

- **Assignee**: Unassigned
- **Comments**: 8/12/24
- **Created**: 8/6/24
- **Description**: When an element is type Reference, the allowed targets are displayed in the rendering (e.g., Reference(Patient)).  However, when an element is type Resource, the rendering does not include restrictions (e.g., Bundle.issues, Bundle.entry.response.outcome), making it appear that any resource is allowed (even though in both of those cases only OperationOutcome is permitted).

This ticket is a request to update the tooling to display the profile on elements defined as Resource so that it is clear which resource types are allowed.
