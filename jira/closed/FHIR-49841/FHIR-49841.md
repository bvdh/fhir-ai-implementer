# FHIR-49841: Add a mention of preferredTerminologyServer to item.answerValueSet

## Metadata

- **Assignee**: Unassigned
- **Created**: 2/19/25
- **Description**: The description of item.answerValueSet could benefit from a mention of the &#91;preferred terminology server extension&#93;(https://www.hl7.org/fhir/extensions/StructureDefinition-preferredTerminologyServer.html) since in order to show that list, in most cases you need to be able to expand the ValueSet URL.  Some suggested text to add:

"If the ValueSet requires a terminology server to provide an expansion, a preferred terminology server can be provided with an &#91;extension&#93;(https://www.hl7.org/fhir/extensions/StructureDefinition-preferredTerminologyServer.html) on the Questionnaire item."
- **Grouping**: [SDC-CORE]
- **Issue Type**: Change Request
- **Key**: FHIR-49841
- **Related Artifact(s)**: Questionnaire
- **Related URL**: https://www.hl7.org/fhir/questionnaire-definitions.html#Questionnaire.item.answerValueSet
- **Reporter**: Paul Lynch
- **Resolution**: Persuasive with Modification
- **Resolution Description**: "If the ValueSet requires a terminology server to provide an expansion, a preferred terminology server can be provided with an [extension]([https://www.hl7.org/fhir/extensions/StructureDefinition-preferredTerminologyServer.html)] on the Questionnaire item, an ancestor item, or the Questionnaire root."
- **Status**: Applied
- **Summary**: Add a mention of preferredTerminologyServer to item.answerValueSet
