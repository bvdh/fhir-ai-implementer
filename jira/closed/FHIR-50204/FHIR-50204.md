- **Resolution Description**: Will add a section on the search page called "Search for 'additional' resources" 

"There is no difference between searching for 'standard' resources vs. [additional](resource.html#additional) resources.  The specification that defines the additional resource *SHOULD* define a set of standard search parameters for that resource, though other specifications are free to define more.

Clients may need to deal with unanticipated 'additional' resources, either injected as unrequested _include or _revinclude resources provided by the server, or as [contained] resources within the returned entries."
