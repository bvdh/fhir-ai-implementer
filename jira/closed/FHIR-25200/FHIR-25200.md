# FHIR-25200: text/json is not a valid mimetype

- **Description**: Section 3.1.0.1.9 states:

         "If a client provides a generic mime type in the Accept header (application/xml, text/json, or application/json), the server SHOULD respond with the requested mime type, using the XML or JSON formats described in this specification as the best representation for the named mime type (except for binary - see the note on the Binary resource). "

         But according to the list of media types by IANA (http://www.iana.org/assignments/media-types/media-types.xhtml#text), text/json is not a valid mediatype at all.

         And text/xml is a valid media type, but is not mentioned in the section.

         Suggestion: change text/json to text/xml.
