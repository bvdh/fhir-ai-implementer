# FHIR-53850: Not all CanonicalResource search parameters are defined in all implementations

## Metadata

- **Assignee**: Unassigned
- **Created**: 1/7/26
- **Description**: CanonicalResource defines "search parameters defined by this interface for all implementations".

However, as can be seen at https://build.fhir.org/searchparameter-registry.html#canonicalresource, these parameters are not always defined for all canonical resources.

	none of the "context" params, "description", nor "jurisdiction" are defined for any of
	
		SubscriptionTopic
		DeviceDefinition
		ObservationDefinition
		SpecimenDefinition
	
	
	All other parameters, except "url", seem to have at least one resource missing a definition
