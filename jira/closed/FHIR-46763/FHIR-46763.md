- **Description**: This text in the description of the _maxresults search param gives the impression that 'include' resources might count towards the _maxresults limit:



The parameter _maxresults is defined as an instruction to the server regarding the limit on how many total resources should be returned from a query. If supported, a server SHOULD NOT return more resources than requested.

It would be helpful to clarify that _maxresults is intending to limit the number of 'match' results and _include / _revinclude should work as expected against those 'match' resources.
