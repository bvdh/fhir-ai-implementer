- **Description**: The extension https://hl7.org/fhir/extensions/StructureDefinition-resolve-as-version-specific.html has been created such that a transaction can indicate to a server that the given transaction specific reference should be populated with a version specific reference rather than the default non-version reference. A good use of this is when a transaction (Transaction, Batch, other) are requesting that a set of resources be preserved and include a Provenance pointing to each of them; and where the request is to use version specific Provenance.target.
