# FHIR-17001: Suggest that the element fhirVersion 0..1 string be added to resource.meta - 2018-May Core Norm Infrastructure #112

## Metadata

- **Assignee**: Unassigned
- **Created**: 5/13/18
- **Description**: Comment:

         The Resource.meta does not currently contain an element for the recording of the version of FHIR to which the resource conforms. Suggest that the element fhirVersion 0..1 string be added to resource.meta to hold the FHIR version and that the instruction be for the creator/client/sender of the resource to provide the FHIR version to which the resource instance conforms.

         Summary:

         Suggest that the element fhirVersion 0..1 string be added to resource.meta
