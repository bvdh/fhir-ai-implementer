# FHIR-55163: Potentially re-useable THO-candidate value sets in resource Group

## Metadata

- **Assignee**: Unassigned
- **Created**: 1/20/26
- **Description**: The Group references several value sets that are potential candidates for migration to THO. While value set migration is not required, if it's realistic for the value set definition to need to change in the coming years, or the value sets might be useful in CDA, V2 or other specs, migration should be considered. Specifics: CodeSystem: group-code (GroupType) referenced by ValueSet group-code bound to element Group.code; CodeSystem: definition-method (DefinitionMethod) referenced by ValueSet definition-method bound to element Group.characteristic.method; CodeSystem: characteristic-offset (CharacteristicOffset) referenced by ValueSet characteristic-offset bound to element Group.characteristic.offset; CodeSystem: group-involvement (GroupInvolvement) referenced by ValueSet group-involvement-set bound to element Group.member.involvement;
