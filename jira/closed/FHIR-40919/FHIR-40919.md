# FHIR-40919: Primitives section should include a link to extensions, like examples

## Metadata

- **Assignee**: Unassigned
- **Created**: 4/24/23
- **Description**: The paragraph introducing the primitive data types includes a hyperlink to the primitive examples, but there's no similar link to the extensions tab for primitives.  There should be.  Also, both links should be clear that they're extensions and examples for primitives as opposed to being generic links.
- **Issue Type**: Change Request
- **Key**: FHIR-40919
- **Related Page(s)**: Datatypes
- **Reporter**: Lloyd McKenzie
- **Resolution**: Persuasive with Modification
- **Resolution Description**: We will change the sentence "See also the [Examples|https://build.fhir.org/datatypes-examples.html#primitives]." to "See also the [Primitive Examples|https://build.fhir.org/datatypes-examples.html#primitives] and [Primitive Extensions]."

On the primitive examples section, we will add a tab that shows examples of the primitives in JSON as well as XML.
- **Status**: Applied
- **Summary**: Primitives section should include a link to extensions, like examples
- **Work Group**: FHIR Infrastructure

- **Jira**: https://jira.hl7.org/browse/FHIR-40919
