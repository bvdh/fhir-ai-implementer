- **Description**: Currently OperationDefinition.synchronicity is a code type element with a required binding to http://hl7.org/fhir/ValueSet/synchronicity-control, which is defined in the extensions pack.

 

We should either change the element (type / binding) or move the value set from the extensions pack into core.

 

We should also run a check to see if there are any other places where this happens.
