- **Description**: Event should allow both a CodeableConcept and a Reference, as there's a need to be able to say things like "after admission" or "after discharge" with respect to a particular Encounter or "after enrollment", "after treatment start", or "after treatment end" for a ResearchStudy.

Also, we need to define an initial set of event codes, including those above plus "date of birth" (making offset an age) as well as "deceased date"

We need to make clear that if offset is negative, it expresses a duration prior to the event.  E.g. "3 hours before the patient died".
