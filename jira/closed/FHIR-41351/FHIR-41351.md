- **Description**: in FHIR [specification|http://build.fhir.org/structuredefinition-operation-snapshot.html] for `$snapshot` operation it is said that

```

This is not an idempotent operation

```

But it is an oversight, as snapshot generation is dependent on profiles that are referenced - if those profiles change, the snapshot can change. 

Please, change this sentence to something like this:

```

This is an idempotent operation. Snapshot generation is dependent on profiles that are referenced - if those profiles change, the snapshot can change. For a frozen package, $snapshot should be idempotent.

```
