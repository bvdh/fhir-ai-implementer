# FHIR-44611: Scheduling workflow resources could use a better overview

## Metadata

- **__EMPTY**: Brian Postlethwaite
- **__EMPTY_1**: Public
- **__EMPTY_2**: PA Jan 2024 WGM:

There is an existing section in the adminsitration module with the relationships

https://build.fhir.org/administration-module.html#sched

And also a confluence page describing the many varied approaches that are found "in the wild"

https://confluence.hl7.org/display/PA/Appointment+Scheduling+-+Industry+Divergence

PA recommend updating the workflow page to refer to the administration section above.

- **Comments**: 1/30/24
- **Created**: 1/30/24
- **Description**: I've seen from practice that folks have a hard time understanding the boundaries between the Appointment, Slot, and Schedule resources. It would be great if a diagram (for the image-oriented) and some prose was added to the workflow module page that explained the relationships.

A good example is the finance module which does it well: https://build.fhir.org/financial-module.html
