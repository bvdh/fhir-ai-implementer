# FHIR-47537: Require chained searches into contained resources

- **Description**: The search page currently reads as follows:

"However, chained parameters will be evaluated inside contained resources."

This implies that this support is required, but many FHIR servers do not support it. Please change to:

"However, chained parameters SHALL be evaluated inside contained resources."
- **Issue Type**: Change Request
- **Key**: FHIR-47537
- **Related Page(s)**: Search
- **Related URL**: https://hl7.org/fhir/6.0.0-ballot2/search.html#contained
- **Reporter**: Rick Geimer
- **Resolution**: Persuasive with Modification
- **Resolution Description**: Will change "However, chained parameters will be evaluated" to "However, chained parameters SHOULD be evaluated"

We would like to make this SHALL, but given that many systems don't and this section is already normative, it's too late to do so now.
