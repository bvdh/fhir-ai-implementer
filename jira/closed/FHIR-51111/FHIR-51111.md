- **Description**: Based on FHIR-51025, this proposed enhancement to the TestScript resource will remove the path elements in favor of just the expression element. Also, in order to support multiple path expression languages the existing expression elements data type will be updated to Expression

The expression elements will be documented to minimally support the following languages: text/fhirpath, text/xml and application/json.
