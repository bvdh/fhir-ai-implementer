# FHIR-39526: Validate Operation should have more parameters defined

## Metadata

- **Description**: The validator supports many parameters for controlling validation:

in addition, there are many other parameters and some of them should be added to this operation. e.g. level, language, html-in-markdown, no_unicode_bidi_control_chars
