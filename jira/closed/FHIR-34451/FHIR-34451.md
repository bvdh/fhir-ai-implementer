- **Description**: R4B contains an extension definition http://hl7.org/fhir/StructureDefinition/structuredefinition-standards-status

The extension has a code value with a required binding to http://hl7.org/fhir/ValueSet/standards-status ...however this valueset doesn't exist in R4B.

It does exist at terminology.hl7.org, but with a different URL: http://terminology.hl7.org/ValueSet/standards-status (terminology.hl7.org instead of just hl7.org/fhir).

Should this valueset be in R4B? Or should this extension definition in R4B get updated to use the terminology.hl7.org url for this valueset?
