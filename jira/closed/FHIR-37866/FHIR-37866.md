- **Description**: It seems that starting with R4B the operation-outcome valueset is gone! Yet R4B and the CI build both are calling for the FHIR defined operation-outcome valueset.

	http://hl7.org/fhir/ValueSet/operation-outcome



This valueset appears in R4 publication. It does not appear in R4B.

I presume this was broken when codes were moved to THO. The codeSystem is on THO

	https://terminology.hl7.org/3.1.0/CodeSystem-operation-outcome.html



But the valueSet seems to have been expected to continue in FHIR core... yet didn't.

 

So, either the valueSet should be moved to THO, and FHIR OperationOutcome link to THO valueSet. OR the valueSet appear in FHIR core like it did in R4.
