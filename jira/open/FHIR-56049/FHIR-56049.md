# FHIR-56049: Additional bindings removal in the HL7 Europe Core profiles

## Metadata

- **Assignee**: Unassigned
- **Created**: 3/12/26
- **Description**: Please consider removal of additional bindings in the HL7 Europe Core profiles:

	http://hl7.eu/fhir/base/StructureDefinition/allergyIntolerance-eu-core
	http://fhir.ehdsi.eu/core/StructureDefinition/condition-myhealtheu-core
	http://fhir.ehdsi.eu/core/StructureDefinition/immunization-myhealtheu-core
	http://fhir.ehdsi.eu/core/StructureDefinition/medicationRequest-myhealtheu-core



When defining MyHealth@EU Core profiles, it is not possible to remove additional bindings, which can be confusing for the end user, as additional bindings usually duplicate information provided in the main binding of the data element for MyHealth@EU.
