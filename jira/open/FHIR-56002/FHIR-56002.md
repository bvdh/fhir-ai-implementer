# FHIR-56002: Attachment Thumbnail

## Metadata

- **Assignee**: Unassigned
- **Created**: 3/3/26
- **Description**: When an Attachment is large, there can be the desire to include a thumbnail / preview version. Currently, the only standard representation of a thumbnail is a flag on `DocumentReference.content` (https://build.fhir.org/ig/HL7/fhir-extensions/StructureDefinition-documentreference-thumbnail.html ), which signifies that content is a thumbnail..

Request defining a standard extension on `Attachment`:

	Id: `thumbnail`, `preview`, or `low-resolution`
	value&#91;x&#93;: 1..1 `Attachment`
