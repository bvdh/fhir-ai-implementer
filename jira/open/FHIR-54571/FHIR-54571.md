- **Created**: 1/16/26
- **Description**: The Version Diff tab lists the following differences (for R4/R4B to R6 and R5 to R6):

	Condition.clinicalStatus: Remove codes recurrence, relapse, remission, resolved
	Condition.verificationStatus: Remove codes provisional, differential



This is incorrect, as each of those codes remains in the bound value sets for R6. Is this possibly an error in the tooling that generates the diffs? I've noticed that all of the codes flagged as "removed" are child codes (i.e., they are level 2 in the code hierarchy).
