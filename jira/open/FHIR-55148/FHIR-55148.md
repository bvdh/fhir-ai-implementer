- **Description**: The CapabilityStatement references several value sets that are potential candidates for migration to THO. While value set migration is not required, if it's realistic for the value set definition to need to change in the coming years, or the value sets might be useful in CDA, V2 or other specs, migration should be considered. Specifics: CodeSystem: simple-format-codes (SimpleFormatCodes) referenced by ValueSet supplemented-mimetypes bound to element CapabilityStatement.format; CodeSystem: restful-security-service (RestfulSecurityService) referenced by ValueSet restful-security-service bound to element CapabilityStatement.rest.security.service; CodeSystem: message-transport (MessageTransport) referenced by ValueSet message-transport bound to element CapabilityStatement.messaging.endpoint.protocol;
