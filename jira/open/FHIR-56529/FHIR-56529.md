# FHIR-56529: Allow cqf-resourceType to be used on uri

## Metadata

- **Assignee**: Unassigned
- **Created**: 4/26/26
- **Description**: The cqf-resourceType extension is used to allow the type of the target of a canonical reference to be specified on the reference. Since artifact-uriReference allows referencing non-canonical resources by uri, this extension should be allowed to be used to provide the resource type.

As well, the display extension should be allowed to be used on a uri to provide the display of the reference.
