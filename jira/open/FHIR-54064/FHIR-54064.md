# FHIR-54064: Clarify base64 encoding in Json Signature Requirements

## Metadata

- **Assignee**: Unassigned
- **Created**: 1/13/26
- **Description**: Current text:

The Signature.data is base64 encoded JWS-Signature as specified in RFC 7515: JSON Web Signature (JWS)  

 

Comment:

RFC 7515 specifies that the signature is base64Url encoded. Since signature.data is a base64Binary data element, this implies that we need to base64 encode something that is base64url encoded. Either way, base64 encoded is not what is specified in the RFC, and we could use a little more clarification on what we want to do to get from there to into signature.data.
