# FHIR-53091: acme.org is not a safe example URI

## Metadata

- **Assignee**: Unassigned
- **Created**: 10/9/25
- **Description**: Examples on the search page Search - FHIR v6.0.0-ballot3 use the uri http://acme.org

That URI is not reserved for examples and could belong to a real organisation.

Use http://example.org or similar instead.
