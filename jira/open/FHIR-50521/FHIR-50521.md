# FHIR-50521: TestScript constraints tst-1, tst-2, tst-3 have incorrect expressions

## Metadata

- **Assignee**: Unassigned
- **Created**: 4/30/25
- **Description**: The expressions for the TestScript constraints tst-1, tst-2, and tst-3 do not seem to match their descriptions.

tst-1 and tst-2 claim that only 1 of its 3 fields can be present. But the expression uses xor such that the expression will be true if all 3 fields are present (TRUE xor TRUE xor TRUE = TRUE)
 
Similarly, tst-3 claims that only 1 of its 3 fields can be present, but it is actually only checking if at least 1 is absent.
