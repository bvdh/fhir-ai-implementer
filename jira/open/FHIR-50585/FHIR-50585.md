# FHIR-50585: Constraint qty-4 should check for UCUM system

## Metadata

- **Assignee**: Unassigned
- **Created**: 5/1/25
- **Description**: The expression for qty-4 should also check that the code system is actually UCUM.

Current:

code.exists() implies code.matches('\\\{.*?
}').not()

Proposed:

(code.exists() and system = %ucum) implies code.matches('\\\{.*?
}').not()
