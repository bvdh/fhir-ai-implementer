# FHIR-51491: The work group extension's binding should be changed to the THO value set for HL7 workgroups; and the extensions pack value set and code system should be deprecated or retired

## Metadata

- **Assignee**: Unassigned
- **Created**: 7/14/25
- **Description**: THO contains an HL7 Workgroup code system and corresponding value set.  This value set should be bound to the StructureDefinition Work Group extension's value as the THO code system and value set can more easily and more regularly be updated with changes to HL7's work groups and other committees, compared to the extensions pack.

Further, it removes duplication and ensures that there is only one HL7-maintained code system for these concepts. If the binding is changed, the code system and value set in the extensions pack should be deprecated or retired with a note in their descriptions referring implementers to the THO alterntives.
