- **Description**: In general, we have indicated that there are 4 general categories of IGs and profiles:

	national base profile - not related to use case at all
	domain profile, all about clinical concepts and nothing about system
	workflow spec, describes how exchange is orchestrated
	provenance related
	system profile - actually integrates all the perspectives



It would be useful if authors could indicate what their intent is from these 5 at least (though there might be more). This could go in .usage, but we haven't defined any appropriate terminology for that, so no one knows how to. So we should
