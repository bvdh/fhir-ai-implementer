QuestionnaireResponse
- **Reporter**: Paul Lynch
- **Resolution**: Persuasive
- **Resolution Description**: Will add documentation as follows:

Questionnaire.item - The order items are listed indicates the order in which they should be presented to the user (if they are enabled)

Questionnaire.item.answerOption - The order of the options indicates the order in which they should be presented to the user by default.  Extensions may drive changes to the presentation order.

QuestionnaireResponse.item - The order of the items is the order in which the questions were presented to the user when completing the Questionnaire.
