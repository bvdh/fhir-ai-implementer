# FHIR-50924: The 'HL7 Workgroup Value Set' and 'hl7 workgroup code system' should be deprecated or retired and references changed to the the artefacts in THO

## Metadata

- **Assignee**: Unassigned
- **Created**: 5/16/25
- **Description**: We currently have two nearly identical HL7 Work Group CodeSystem/ValueSet pairs in the extensions pack and in THO:

FHIR Extensions Pack

	http://hl7.org/fhir/extensions/CodeSystem-hl7-work-group.html
	http://hl7.org/fhir/extensions/ValueSet-hl7-work-group.html



THO:

	https://terminology.hl7.org/CodeSystem-hl7-work-group.html
	https://terminology.hl7.org/ValueSet-hl7-work-group.html



 

HL7 and the broader community would be better served if we had a single pair which can be maintained on an ongoing basis.

This tracker item proposes that:

	The 'HL7 Workgroup Value Set' and 'hl7 workgroup code system' in the FHIR Extensions Pack be retired (or deprecated).
	References in artefacts in the extension pack e.g. the WG extension (http://hl7.org/fhir/StructureDefinition/structuredefinition-wg) to the retired (or deprecated) artefacts in 1 should be changed to the ones in THO.
