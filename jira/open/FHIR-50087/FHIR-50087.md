- **Resolution Description**: For ActorDefinition.type:
1. We will change the binding to person | system | collective | other and leave as required and defined in FHIR
2. We will add a new element "category" 0..* CodeableConcept with a 'preferred' binding to a value set that excludes the 'system' code and makes 'collective' abstract.  Will move that code system to the THO spec.
