# FHIR-57374: Update Guidance on Text Linking

## Metadata

- **Assignee**: Unassigned
- **Created**: 5/20/26
- **Description**: The current "Narrative" content under "Internal Id References" refers to the usage of id/idref. As reviewed with FHIR-I, the use of idref is not schema conformant with XHTML and this has resulted in the creation of a new extension of textLink (https://hl7.org/fhir/extensions/StructureDefinition-textLink.html). 

 

Now that the 5.3.0 extension package is published, I recommend updating the text here to refer to this pattern of textLink references. Images/text have also been included in IPS 2.0.0 https://hl7.org/fhir/uv/ips/Design-Conventions.html#linking-narrative-to-discrete-fhir-resources if helpful in consdering updated text. 

I also propose moving the two sections on Internal Id references and Linking Narrative and Data to be Next to each other. 

Signficant history of discussiosn related to the topic are included under https://jira.hl7.org/browse/FHIR-43316
