- **Description**: http://build.fhir.org/languages.html##term references the code display from the CodeSystem http://terminology.hl7.org/CodeSystem/designation-usage

Resolving this leads to https://terminology.hl7.org/1.0.0/CodeSystem-designation-usage.html – note the version: 1.0.0 whereas the current HL7 Terminology IG is 3.1.0.

Furthermore, this code system is not included in 3.1.0 and this code is apparently superseded by preferredForLanguage from the code system http://terminology.hl7.org/CodeSystem/hl7TermMaintInfra

If this is what should be used going forward then the FHIR Spec should be updated to point to preferredForLanguage going forward and some statement should be made about display and backward compatibility.
