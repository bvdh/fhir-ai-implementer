# FHIR-54919: fml mapping tutorial has invalid fhirpath that doesn't have the surrounding brackets

## Metadata

- **Assignee**: Unassigned
- **Created**: 1/20/26
- **Description**: There are many cases where the embedded fhirpath expressions are not contained in brackets as the language requires:

src.a2 as a where a2.length &lt;= 20 -&gt; tgt.a2 = a; // ignore it

should be 

src.a2 as a where (a2.length &lt;= 20) -&gt; tgt.a2 = a; // ignore it
