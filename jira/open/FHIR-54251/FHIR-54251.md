# FHIR-54251: text element missing status

## Metadata

- **Assignee**: Unassigned
- **Created**: 1/14/26
- **Description**: https://hl7.org/fhir/6.0.0-ballot4/narrative.html#linking

The XML sample provided seems missing status in the text field under section in the Composition : 

 


     &lt;section&gt;
        ...
        &lt;text&gt;
          &lt;div xmlns="http://www.w3.org/1999/xhtml"&gt;
            ...
            &lt;p&gt;There is a history of &lt;span id="a1"&gt;Asthma&lt;/span&gt;&lt;/p&gt;
            ...
          &lt;/div&gt;
        &lt;/text&gt;
      &lt;/section&gt;


should be

 


      &lt;section&gt;
        ...
        &lt;text&gt;
           &lt;status value="generated"/&gt;
           &lt;div xmlns="http://www.w3.org/1999/xhtml"&gt;
            ...
            &lt;p&gt;There is a history of &lt;span id="a1"&gt;Asthma&lt;/span&gt;&lt;/p&gt;
            ...
          &lt;/div&gt;
        &lt;/text&gt;
      &lt;/section&gt;
