# FHIR-47134: Provide Guidance on the expected Resource status

## Metadata

- **Assignee**: Rosemary Hofstede
- **Created**: 9/6/24
- **Description**: FHIR defines the following names for functional lists:

$current-problem

$current-medications

$current-allergies

$current-drug-allergies
 

For each of these please define the 'status' you would expect a server to include. For *

	Condition.clinicalStatus - are all appropriate?
	MedicationState.statement.status - recorded? 
	....



When I read this I fears systems will construct quite different lists without additional guidance. At minimum, provide link to status elements.
