# FHIR-53093: Set definition-method CodeSystem to trial use, not draft

## Metadata

- **Assignee**: Unassigned
- **Created**: 10/9/25
- **Description**: Warnings in the EBMonFHIR Implementation Guide include "Reference to draft CodeSystem http://hl7.org/fhir/definition-method|6.0.0-ballot3".  This appears to come from the following extension in this CodeSystem defined in the FHIR specification:
  {
    "url" : "http://hl7.org/fhir/StructureDefinition/structuredefinition-standards-status",
    "valueCode" : "draft"
  }


The standards status for this CodeSystem should be at least "trial-use" if part of the core FHIR specification.
