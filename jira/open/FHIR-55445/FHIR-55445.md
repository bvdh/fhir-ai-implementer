# FHIR-55445: Remove deprecated StructureDefinition.keyword

- **Description**: This is a child ticket created from https://jira.hl7.org/browse/FHIR-53722   Please see that ticket for broader discussion of the issue.  

The element listed in the name of this issue was intended to be deprecated.  You can consider these guidelines when determining whether it is appropriate to remove it in R6:

	If the element has had the deprecated flag since R4, remove the element.
	If the element has a comment in its description indicating that it is deprecated and one or more of the following is true, remove the element:
	
		it has a trial use flag
		the resource was trial-use in R4
		the element was added in R5 or later
	
	
	If the element has a comment in its description indicating that it is deprecated_ but does not meet any of the criteria in the bullet above, add the deprecated flag to that element.
