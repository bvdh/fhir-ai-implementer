The jurisdictional databases of medical devices (GUDID for US, EUDAMED for EU) define 3 complimentary UDI identifiers for medical devices, that need to be recorded in resources DeviceDefinition and Device for representing devices in catalogs, using the identifier 0..* Identifier element. These 3 complimentary UDI identifiers are the "Secondary UDI-DI", the "Direct marking UDI-DI" and the "Unit of use DI". Their intended representation with the Identifier data type is depicted as follows:

Types DIRECT-MARKING and SINGLE-ITEM are new values proposed by this tracker for addition to ValueSet identifier-type and to CodeSystem v2-0203. They are defined in the table below, which also proposes an addition (text in bold) to the definition of type UDI:
