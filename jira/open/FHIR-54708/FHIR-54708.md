We have seen the need to pass some parameters to the map, such as timestamp. For example to populate a Provenance resource, or simply have the time of the data transformation in the resource somehow.

We should explain how this can be achieved with the current operation definition (e.g. with Parameters?) or provide a mechanism to pass key-value pairs into the operation.
