# FHIR-50589: The example for _list should not escape the $

## Metadata

- **Assignee**: Unassigned
- **Created**: 5/1/25
- **Description**: On search.html, the example for _list escapes the $ with a \ :

GET &#91;base&#93;/AllergyIntolerance?patient=42&amp;_list=\$current-allergies

This doesn't match the example at http://hl7.org/fhir/6.0.0-ballot3/lifecycle.html#current
I assume that the escaping on search.html is the incorrect one.
