- **Description**: Many patient monitoring customers use VisitNumber as a patient identifier.  FHIR does not have a direct mapping in https://build.fhir.org/valueset-identifier-type.html e.g. for the Patient resource.  Closest mapping is the Encounter resource but it's identifier is not specifically a VisitNumber. 

Request either  1.  Encounter resource expand the description for identifier to allow for VisitNumber to avoid customer confusion or  2.  Add VisitNumber to https://build.fhir.org/valueset-identifier-type.html so it can be referenced in the Patient resource e.g. like MRN, etc.
