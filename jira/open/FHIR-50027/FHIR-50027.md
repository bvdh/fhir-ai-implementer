- **Description**: The cqf-relativeDateTime and relative-date extensions have both been deprecated in favor of the new relative-time extension.

This extension is defined based on the new R5 TimeFromEvent data type:

https://github.com/HL7/fhir-extensions/blob/master/input/definitions/datatypes/StructureDefinition-relative-time.xml

However, due to the inability of the current tooling to make use of the R5 data in the R4 space, this extension is still commented out. This tracker is to get this relative-time extension incorporated into extensions.
