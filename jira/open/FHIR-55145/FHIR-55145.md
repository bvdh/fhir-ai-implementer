# FHIR-55145: Bundle.type has no escape valve

## Metadata

- **Assignee**: Unassigned
- **Created**: 1/20/26
- **Description**: The element Bundle.type is a mandatory element with a required binding, but appears to have neither an 'unknown' or 'other' concept, which can make the containing element impossible to populate in some cases.
