# FHIR-53987: Consider using inline quote for vertical pipe ( | ) and corresponding URL-encoded value

## Metadata

- **Assignee**: Unassigned
- **Created**: 1/12/26
- **Description**: when describing the format of token parameters, the vertical pipe character (

|

) is used instead of the URL-Encoded value of

%7C

.
Consider whether representing the vertical pipe character and its corresponding URL-encoded value as an inline quote would enhance readability.

Additionally, the sentence is not capitalized.
