- **Description**: How can a structuremap transform parameter support just having whitespace in it?

This is not uncommon for things like:

v.prefix as pre, v.linkId as v -&gt; tn.text = append(prefix, ' ', v) "r1";

Doing this generates errors as content can't be whitespace only in fhir.
In the HAPI serializer it drops the element entirely and you end up with null in the output.
