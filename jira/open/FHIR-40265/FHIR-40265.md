# FHIR-40265: Enable content from detailed descriptions to be pulled into content

- **Description**: It is common that a user wants to describe in their content section of a guide a summarized form of details discussed in the detailed descriptions of a guide. In these scenarios, IG authors often repeat that content, so it can be out of sync or different - see example issue https://jira.hl7.org/browse/FHIR-39393. (Related URL above shows this example)

We ought to enable IG authors to reference content in the detailed descriptions (such as the Definition, but perhaps also the Note or other content) to be pulled into the content section. This way the IG author can write text once and present it in another format on their content page as is useful.
