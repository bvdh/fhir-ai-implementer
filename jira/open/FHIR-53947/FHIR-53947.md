	The json 'resourceType' property is not present in the object model - that is the type of the object (can restrict to specific types using `ofType(Patient)`)

	There is no difference between an attribute and an element in XML (url and id are just considered properties in the object model)
	Primitive FHIR types do not have a `value` property, but they do have id and extension properties
