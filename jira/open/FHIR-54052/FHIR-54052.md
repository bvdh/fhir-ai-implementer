# FHIR-54052: Typo in _summary section of Search page

## Metadata

- **Assignee**: Unassigned
- **Created**: 1/13/26
- **Description**: There is a typo in the following sentence:
"Servers SHOULD mark the by populating meta.tag resources with the code SUBSETTED   to ensure that the incomplete resource is not accidentally used to overwrite a complete resource".
- **Issue Type**: Technical Correction
- **Key**: FHIR-54052
- **Related Page(s)**: Search
- **Related Section(s)**: 3.2.1.7.5
- **Related URL**: https://hl7.org/fhir/6.0.0-ballot4/search.html#_summary
- **Reporter**: DanielSchuster
- **Resolution**: Persuasive
- **Resolution Description**: Change the test:

Servers SHOULD mark the by populating {{meta.tag}} resources with the code [{{SUBSETTED}}  !https://hl7.org/fhir/6.0.0-ballot4/external.png!|https://terminology.hl7.org/CodeSystem-v3-ObservationValue.html#v3-ObservationValue-SUBSETTED] to ensure that the incomplete resource is not accidentally used to overwrite a complete resource.

to:

Servers SHOULD indicate the resource as potentially incomplete by populating {{meta.tag}} with the code [{{SUBSETTED}}  !https://hl7.org/fhir/6.0.0-ballot4/external.png!|https://terminology.hl7.org/CodeSystem-v3-ObservationValue.html#v3-ObservationValue-SUBSETTED] to ensure that the incomplete resource is not accidentally used to overwrite a complete resource.
