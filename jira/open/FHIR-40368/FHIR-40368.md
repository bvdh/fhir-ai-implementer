# FHIR-40368: Link to structuredefinition.html#logical is ambiguous

## Metadata

- **Assignee**: Unassigned
- **Created**: 1/31/23
- **Description**: In both R4 and R5, there are two #logical anchors on structuredefinition.html.

In R5, the first is in section 5.4.6.4 Common Mapping Targets, in the "Logical mapping" row.

The second is the section 5.4.6.5 Logical Models.

The same behavior happens in R4.

This matters because the link with the arrow below points at structuredefinition.html#logical, and it's not clear which part of the page the user should be looking at.
