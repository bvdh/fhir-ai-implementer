# FHIR-55680: Not all data types have the "used in the following places:" data

## Metadata

- **Assignee**: Unassigned
- **Created**: 2/10/26
- **Description**: Complex data types on the datatype page include information on where the data type is used (a list of resources containing data elements using the data type). However, at least a few data types which are documented on other pages (specifically the References and Narrative page) don't have this information. It would be nice if all data types had this information.
