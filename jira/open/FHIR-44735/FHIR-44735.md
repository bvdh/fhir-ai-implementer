- **Description**: It was agreed at the January 2023 WGM that the following value sets would be moved to THO:

ucum-common

ucum-units - defined as "all codes" - will make immutable

and the following would just be removed from the core spec without a THO replacement:

all-distance-units

all-time-units

A UP ticket has already been created for ucum-common and ucum-units and will be in the next release of THO.

FHIR-I agreed that they would remove the four value sets from the core spec and update any pointers to ucum-common and ucum-units to point to the THO value sets.

This ticket is for that removal.
