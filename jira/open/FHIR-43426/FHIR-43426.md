# FHIR-43426: Explain the use of "?" in Differential Elements on Detailed Description Tab

## Metadata

- **Assignee**: Unassigned
- **Created**: 12/28/23
- **Description**: If you go to Detailed Description tab on any profile, the upper cardinality is represented by "?".  The conformance rules do not explain what this means on cardinality.  A definition should be added so as not to confuse the user with the "?!" flag.

Examples of where this can be found
http://build.fhir.org/ig/HL7/US-Core/StructureDefinition-us-core-allergyintolerance-definitions.html - Differential Elements Table tab
http://hl7.org/cda/us/ccda/2024Jan/StructureDefinition-AllergiesAndIntolerancesSection-definitions.html - Differential Elements Table tab
- **Issue Type**: Change Request
- **Key**: FHIR-43426
- **Related Page(s)**: Conformance Module
- **Related Section(s)**: 2.1.1.0.3
- **Related URL**: https://build.fhir.org/conformance-rules.html#cardinality
- **Reporter**: Linda Michaelsen
- **Resolution**: Persuasive with Modification
- **Resolution Description**: Will provide an explanation of what the ? means here: [https://build.fhir.org/ig/FHIR/ig-guidance/readingIgs.html] (which is what IGs should be pointing to for help in understanding profile rendering).

Specifically, "?" in a differential means that the cardinality has not been changed from whatever it was in the parent specification.
